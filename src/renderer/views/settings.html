How to Get a Faceit API Key

  1. Go to the Faceit Developer Portal: developers.faceit.com.
  2. Log in with your Faceit account.
  3. Open the App Studio
  4. Create a new API application by clicking "Create New App".
  5. Fill out the required fields (name, description, etc.) and submit.
  6. Click on the newly created application.
  7. Head to the API section.
  8. Create a new client-sided API key.
  9. Copy your API key and paste it into the field in this app.
